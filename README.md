# Hattrick Team Assistant

An open-source, locally-run analytical helper for Hattrick managers. Built originally for my own three teams under the manager account `robi_` (Hetvehely 158111, Wetzikon 2839340, Apa lanyai 3235631), published so other technically-minded Hattrick managers can clone, configure, and run it on their own machines.

## What it does

Read-only access to Hattrick CHPP API endpoints to power:

- League standings + opponent scouting
- Lineup recommendations
- Financial projections (income, costs, bankruptcy planning)
- Training plans + skill pop estimates
- Match preparation reports

Analysis is generated by a large language model from the retrieved CHPP data.

## Architecture

Local Python application. Each user registers their own Hattrick CHPP application (free, requires HT staff approval), obtains their own consumer key + access tokens, and runs Team Assistant on their own machine. The repository does not contain shared credentials, no central server, no shared user database.

## Scope

- Read-only, does not modify any game state in Hattrick
- Rate-limited to a few requests per minute, well within CHPP guidelines
- Local-only, no public endpoint, no web service, no telemetry
- Each user holds their own credentials and uses them only against their own account

## CHPP endpoints requested

Public:
- `teamdetails`, `worlddetails`
- `leaguedetails`
- `matches`, `matchdetails`
- `players` (public view), `arenadetails`

Private (own teams):
- `economy`
- `playerdetails`, `training`
- `clubdetails`, `lineup` (read)
- `transfersteam`, `youthplayerlist`

## Tech stack

- Python 3
- `requests-oauthlib` for OAuth 1.0a
- Local module + per-session cache, optional disk cache for repeat queries

## Self-hosting (for other managers)

1. Apply for your own Hattrick CHPP application at the Hattrick CHPP Manager page (free, ~2-3 week approval). Request the same read-only feature set listed above.
2. Once approved, copy your consumer key and consumer secret.
3. Clone this repository, copy `.chpp-credentials.example.json` to `.chpp-credentials.json`, fill in your keys.
4. Run the one-time OAuth bootstrap (`python -m team_assistant.auth`) to authorize the app against your Hattrick account.
5. Bring your own LLM API key (Anthropic or OpenAI), set it in the same config file.
6. Run analyses against your own teams. The tool will never touch any account other than the one you authorized.
